
开发一个简单的备忘录应用，包含以下功能：

- 登录页面：用户名和密码输入，记住登录状态
- 信息列表页面：显示所有备忘录信息条目
- 信息详情/编辑页面：查看和编辑备忘录信息

技术要点

1. 界面实现

- 使用ConstraintLayout实现登录和详情页面布局
- 使用RecyclerView实现信息列表
- 实现合理的页面导航和过渡动画

2. 功能实现

- 使用SharedPreferences存储登录状态和应用设置
- 使用SQLite数据库存储个人信息数据
- 实现信息的增删改查操作
- 添加适当的用户交互反馈（Toast、Dialog等）

3. 进阶要求（可选）

- 实现数据备份与恢复功能
- 添加信息搜索功能
- 实现信息分类和排序功能
- 添加图片选择和显示功能
